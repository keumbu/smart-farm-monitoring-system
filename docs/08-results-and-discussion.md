# 08 - Results and Discussion

## Overview

This chapter presents the expected outputs, simulated results, and sample data generated by the Smart Farm Monitoring System. Since the project is currently under active development, the results are based on simulations and prototype-level testing.

The goal is to demonstrate how the system collects, processes, and presents agricultural data for monitoring and decision-making.


## Expected Outputs

The system is designed to provide continuous monitoring of environmental and soil conditions.

Expected outputs include:

- Air temperature readings
- Relative humidity levels
- Soil moisture values
- Soil temperature readings
- Light intensity measurements
- Rainfall detection status
- Device connectivity status
- Historical data trends
- Alert notifications

The simulated results confirm that the system is capable of:

- Collecting environmental sensor data
- Processing and formatting data on ESP32
- Transmitting data via Wi-Fi
- Storing and structuring cloud data
- Displaying information in a readable format

Although the system is still under development, the architecture proves that real-time agricultural monitoring is achievable using IoT technologies.

Future implementation will replace simulated values with real sensor data collected from physical deployment.
